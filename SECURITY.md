## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email: [xabierlameiro@gmail.com](mailto:xabierlameiro@gmail.com)

We will respond within 72 hours and work with you to understand and patch the issue.
